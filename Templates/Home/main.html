About This Project

This AI-powered Posture Detection System helps users identify and correct posture deformities using advanced machine learning and image processing techniques. By analyzing side-profile images, it detects spinal misalignments, forward head posture, and other issues that can lead to long-term discomfort. Using Python and OpenCV, the system evaluates body posture and recommends personalized exercises tailored to improvement.
